# ZenFinance 💎

**ZenFinance** — это современное и минималистичное приложение для управления личными финансами, созданное с акцентом на спокойствие и осознанное потребление.

![ZenFinance Icon](https://picsum.photos/seed/finance/200/200)

## ✨ Основные возможности

- **Умный Дашборд**: Наглядный обзор вашего баланса, доходов и расходов.
- **Цели накопления**: Устанавливайте цели (например, на новый MacBook) и следите за прогрессом в реальном времени.
- **Управление бюджетами**: Устанавливайте лимиты по категориям (Еда, Транспорт, Развлечения и др.), чтобы не тратить лишнего.
- **AI-Инсайты**: Персональные советы от искусственного интеллекта на основе ваших трат.
- **Мультивалютность**: Поддержка KZT, RUB, USD и EUR.
- **Генератор иконок**: Встроенный инструмент для создания уникальных аватаров проекта.

## 🚀 Технологии

- **Frontend**: React + TypeScript + Tailwind CSS
- **Анимации**: Framer Motion
- **Иконки**: Lucide React
- **AI**: Google Gemini API
- **Сборка**: Vite

## 🛠 Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-логин/zen-finance.git
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Создайте файл `.env` и добавьте ваш API ключ:
   ```env
   VITE_GEMINI_API_KEY=ваш_ключ
   ```
4. Запустите проект:
   ```bash
   npm run dev
   ```

## 📱 Мобильная версия
