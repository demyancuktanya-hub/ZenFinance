# 💎 ZenFinance

Умный и минималистичный менеджер личных финансов с поддержкой ИИ (Gemini) и PWA.

![ZenFinance Icon](public/icon.svg)

## ✨ Особенности
- **PWA Ready**: Установите как приложение на iOS или Android с красивой иконкой.
- **ИИ-советы**: Получайте персональные рекомендации по экономии на основе ваших трат.
- **Инвестиции**: Отдельный модуль для отслеживания акций и криптовалют.
- **Бюджеты**: Устанавливайте лимиты по категориям и следите за их выполнением.
- **GitHub Sync**: Встроенная система синхронизации кода.

## 🚀 Быстрый старт (Vercel)
1. Нажмите **Deploy** в Vercel.
2. Выберите этот репозиторий.
3. Добавьте переменную окружения `GEMINI_API_KEY`.
4. Готово!

## 🛠 Технологии
- React + TypeScript
- Tailwind CSS
- Express (Backend)
- SQLite (Database)
- Google Gemini API
- Lucide Icons
